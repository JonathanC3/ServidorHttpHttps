Http Funcionamiento

Las reglas de HTTP son claras HTTP sobre la forma cómo se comunican un cliente y un servidor. Por lo que debe tener en cuenta que solo los clientes pueden realizar solicitudes HTTP a los servidores HTTP, y estos solo pueden responder a la solicitud de un cliente HTTP. Cuando los clientes solicitan un archivo deben proporcionar la URL del archivo, y el servidor debe responder todas las solicitudes HTTP, y si no encuentra la página solicitada, debe hacerlo con un mensaje de error.
